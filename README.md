Sample Python application to bootsrap idea ship with :

* Built-in CLI with fire
* Config file
* Basic logging
* CICD in Gitlab

# 1. HOWTO

## 1.1 Setup virtualenv

```
./setup.sh
```

## 1.2 Ship app into Docker

```
./build.sh
```

## 1.3 Deploy via Docker Compose

```
./run.sh
```
